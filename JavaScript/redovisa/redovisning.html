Kmom06

Under kursmomenten så kom upp termerna scope och closure, jag har redan använt de sedan tidigare, och man kan har nytta av de på olika sätt, då scope är variabler som deklarerade inuti en function, då är dem lokan(privata) variabler och kan nås bara i funktionen eller genom function. det kan vara olika typer av variabler även funktioner. Medan closure då är de variabler som är global då kan den nås från vilken fil samt och tillgängligt i hela projektet, det kan vara alla typ av variabler eller funktioner. Det var inget nytt för mig under kursmomenten, det var bara som en påminnelse om vad jag vet innan. Jag skapade en liknande projekt med då vara det ett spel där man kan skapa en bil och välja egenskaper till bilen som färg och motor etc. Termerna this och instans var också använda under kursmomenten, de är jag väldigt medveten om då jag använde de mycket med mina tidigare java project. “This” termen är som ett sätt för att exportera variabler eller funktioner av en object då man kan kalla på de så man kan sätta eller hämta specifika värde för just den nya variablen, till example:

const test = {

prop: 42,

setProp: function(value) {

return this.prop = value;

},

getProp: function() {

return this.prop;

},

};

test.setProp(12);

console.log(test.getProp()); // Förväntad utgång: 42

Så en instance är vår grund object där kan vi skapa nya reference för och använda objekten. Denna kursmomenten skapade jag hängman spelet, det var första gången jag gör den spelet med javascript, men inga svårigheter som stör mig under tiden, det gick snabbt och smidigt.
